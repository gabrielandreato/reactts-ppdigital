# 🌐 Escola Digital (Front end) - Projeto React com Typescript alimentado por uma API REST Django.

### 📚💻Técnologias e Conhecimentos aplicados:
- Estilização com Css, Grid e Flexbox.
- React
- Typescript
- Recoil
- React-Router-Dom
- NGINX
- Docker
